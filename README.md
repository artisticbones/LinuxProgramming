### 开发相关
* 尽量不要声明很多大变量，如果大的结构体，那么需要动态分配
* 在查看接口API的时候会经常遇到("_ _")的函数，这些函数通常代表接近硬件的函数，如果要使用，请搞清楚在使用。
* 内核编程不支持浮点数运算，开启浮点数运算通常会使内核保存和重新存储浮点运算指针的状态。尽量不要这么做，因为内核的空间很珍贵。
